Implementação Blazor: Explique como você transpôs a hierarquia visual do Miro para os componentes Blazor? 
Eu peguei o layout do Miro e dividi em partes menores, transformando cada uma em um componente no Blazor, como menu, cabeçalho e os cards de gastos. Depois, juntei tudo em uma página principal.

Dificuldade Técnica: Qual foi o maior desafio ao componentizar o GrupoCard?
A maior dificuldade foi fazer o GrupoCard funcionar com dados diferentes (nome, valor, tipo) e mudar o estilo automaticamente (como cores para valores positivos ou negativos).
